GAE Based APIs

Benefits

  • Base config files for GAE and DRF.
  • Setup scripts.
  • Datastore and DRF utils.

How to use

This is a base Google App Engine project that includes Django Rest Framework. In order to use it clone the project, create virtual environment and run configure_requirements.py script. This assumes that you have GAE installed and configured correctly. Running which appcfg.py in terminal should respond with a full path to App Engine's appcfg.py script.
